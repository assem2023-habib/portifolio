Implementation Guide

دليل شامل لتطبيق قسم المهارات المحسّن مع الأيقونات الحقيقية والحركات الانسيابية والألوان الرسمية

📋 ما تم إضافته

✓ أيقونات حقيقية لكل تقنية (Font Awesome icons)

✓ الألوان الرسمية لكل لغة/framework من الـ Brand guidelines

✓ Staggered Animation عند التحميل — كل بطاقة بتأخير مختلف (0.05s - 1.05s)

✓ Smooth Transitions مع cubic-bezier easing

✓ Hover Effects — الأيقونة تكبر وتدور، البطاقة ترتفع

✓ Pulsing Dots — نقاط مستوى الكفاءة تنبض بلطف

✓ Responsive Design — يتكيف تلقائياً مع جميع أحجام الشاشات

🚀 خطوات التطبيق

الخطوة 1: إضافة مكتبة الأيقونات

في ملف HTML الرئيسي، أضف هذا السطر في <head>:

<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css" rel="stylesheet">

هذا يوفر جميع الأيقونات المستخدمة (fab fa-php, fab fa-laravel, إلخ)

الخطوة 2: نسخ ملف CSS

انسخ كود من skills-section-complete.css واللصقه في ملف CSS مشروعك

أو استخدم ملف منفصل وأضفه في <head>:

<link href="assets/css/skills-section.css" rel="stylesheet">

الخطوة 3: تحديث HTML البطاقات

استخدم البنية التالية لكل بطاقة مهارة:

<div class="skill-card skill-php"> <div class="skill-header"> <div class="skill-icon-box"> <i class="fab fa-php"></i> </div> <div class="skill-name"> <h6>PHP</h6> <span class="skill-category">Backend</span> </div> </div> <div class="skill-level"> <span class="level-dot" style="background-color: #28a745"></span> <small class="level-text" style="color: #28a745;">Excellent</small> </div> </div>

لاحظ الـ class skill-php — هذا يحدد الألوان

⏱️ مواصفات الحركات

Staggered Animation (عند التحميل)

  • Duration: 0.6s لكل بطاقة
  • Easing: cubic-bezier(0.34, 1.56, 0.64, 1) — smooth ease-out
  • From: opacity: 0; translateY(30px) scale(0.95)
  • To: opacity: 1; translateY(0) scale(1)
  • Delays: 0.05s, 0.10s, 0.15s ... 1.05s (موجة)

النتيجة: كل بطاقة تظهر بتأخير 50ms أكثر من السابقة — تأثير موجة سلس

Hover Effect

  • Duration: 0.35s
  • Easing: cubic-bezier(0.34, 1.56, 0.64, 1)
  • Transform: translateY(-8px) — ترتفع البطاقة
  • Box Shadow: 0 12px 40px rgba(108,99,255,.2)
  • Icon Scale: scale(1.12) rotate(5deg)

Pulse Animation (نقطة المستوى)

  • Duration: 2s infinite
  • Easing: ease-in-out
  • From/To: opacity: 1 → 0.6 → 1

💡 نصائح مهمة

⚠️ تأكد من إضافة Font Awesome — بدونها، الأيقونات لن تظهر
💾 استخدم Classes الصحيحة — كل تقنية لها class خاص (skill-php, skill-laravel, إلخ) يحدد الألوان
🎯 الأيقونات قابلة للتخصيص — يمكنك تغيير الأيقونات من Font Awesome إلى أيقونات أخرى إذا أردت
🔄 الحركات تحدث تلقائياً — لا تحتاج لـ JavaScript، كل شيء بـ CSS والـ HTML animations
