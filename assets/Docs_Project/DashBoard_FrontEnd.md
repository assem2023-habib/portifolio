## نظرة عامة

**HTML/CSS Template Four** هو قالب لوحة تحكم (Dashboard) متكامل مطوّر باستخدام HTML و CSS فقط، بدون أي مكتبات JavaScript خارجية. مبني على قالب Elzero Web School مع إضافات وتعديلات.

### مكونات المشروع

- **8 صفحات HTML** كاملة التصميم
- **نظام ألوان** متكامل ومتناسق
- **إطار عمل CSS** مساعد (Framework) يحتوي على كلاسات Utility جاهزة
- **تصميم متجاوب** (Responsive) يعمل على جميع الشاشات
- **أيقونات Font Awesome** لتجميل الواجهات

## فوائد المشروع

هذا القالب ليس مجرد تصميم جميل، بل هو **أداة عملية** تقدم فوائد مباشرة للمطورين وأصحاب الأعمال:

### 👨‍💻 للمطورين
- **تعلّم بنية المشاريع الحقيقية**: كيفية تقسيم ملفات HTML و CSS لمشروع كامل (8 صفحات) مع إطار عمل مساعد
- **فهم التصميم المتجاوب**: أمثلة عملية على (Media Queries) لتكييف التصميم مع الموبايل والتابلت
- **أنماط CSS احترافية**: استخدام Flexbox و Grid في تطبيقات عملية، متغيرات CSS، وتقنيات تنسيق متقدمة
- **توفير وقت التطوير**: Framework جاهز يغنيك عن كتابة كلاسيك التنسيق المتكررة

### 🏢 لأصحاب المشاريع والشركات
- **نموذج أولي جاهز**: ابدأ مشروع لوحة التحكم الخاص بك فوراً دون الحاجة لتصميم من الصفر
- **قابلية التوسع**: 8 صفحات تغطي معظم احتياجات أنظمة الإدارة (Dashboard، إعدادات، مستخدمين، مشاريع، تقارير)
- **تكلفة صفرية**: قالب مجاني بمكتبات مفتوحة المصدر (Font Awesome، Google Fonts)

### 🎓 للمتعلمين
- **مرجع تعليمي متكامل**: كل سطر في الكود موثّق بتعليقات بالعربية
- **تطبيق عملي**: كل مفهوم في HTML/CSS تجد له تطبيقاً مباشراً في هذا القالب

## الخدمات التي يقدمها القالب

الخدمات الأساسية الجاهزة التي يمكن لأي موقع أو تطبيق استعمالها فوراً:

### 📊 لوحة المعلومات (Dashboard)
- ملخص الأداء العام مع أرقام سريعة (إجمالي المستخدمين، المشاريع، المبيعات)
- ودجتس تفاعلية: ترحيب، مسودة سريعة، أهداف الشهر مع شريط التقدم
- آخر الأخبار، المهام، الملفات المرفوعة، والتذكيرات

### ⚙️ نظام إعدادات متكامل
- التحكم العام في الموقع (اسم الموقع، الوصف، البريد الإلكتروني)
- إدارة الأمان وكلمة المرور والتحقق بخطوتين
- روابط التواصل الاجتماعي والسوشيال ميديا
- التحكم في ودجتس Dashboard ونظام النسخ الاحتياطي (Backup)

### 👤 نظام المستخدمين والملفات الشخصية
- صفحة ملف شخصي متكاملة مع الصورة الرمزية والمعلومات
- إحصائيات المستخدم (المشاريع، المبيعات، التقييمات)
- أقسام منفصلة: المعلومات العامة، معلومات الاتصال، آخر نشاط

### 📁 إدارة المشاريع
- عرض قائمة المشاريع مع صور الفريق وتقدم العمل
- جدول تفصيلي ببيانات المشاريع (الاسم، المدة، الفريق، الحالة)
- مواعيد نهائية ونسب إنجاز

### 🎓 منصة تعليمية مصغرة
- عرض الكورسات مع صور وأسماء المدربين
- نسبة التقدم، عدد المشتركين، وسعر كل كورس
- تقييمات ومراجعات

### 👥 نظام إدارة الأصدقاء والعلاقات
- بطاقات أصدقاء مع الصورة والمعلومات الشخصية
- إحصائيات (المقالات، الصور، الفيديوهات)
- حالة الصداقة وتاريخ الانضمام

### 📂 إدارة الملفات والوسائط
- عرض الملفات المصورة (PDF, PSD, ZIP, AVI, PNG, EPS, DLL, SQL, TXT)
- أحجام الملفات وتواريخ الرفع
- إحصائيات إجمالية للملفات حسب النوع والحجم

### 💳 نظام الخطط والاشتراكات
- 3 خطط: مجانية، أساسية، متميزة
- مقارنة المميزات بين الخطط
- أزرار تحكم بالاشتراك (Join/Current)

## إطار العمل المساعد (Framework CSS)

ملف `framework.css` يوفر مجموعة من الكلاسات الجاهزة لتسريع التطوير:

### أنظمة العرض
| الكلاس | الوظيفة |
|--------|---------|
| `.d-flex` | تفعيل Flexbox |
| `.d-grid` | تفعيل Grid |
| `.d-block` | عرض كتلة |
| `.center-flex` | توسيط Flexbox كامل |
| `.between-flex` | مسافة بين العناصر |
| `.f-wrap` | التفاف العناصر |

### المسافات (Padding & Margin)
- `p-10`, `p-15`, `p-20` - Padding
- `pt-10`, `pb-15`, `pl-15` - Padding اتجاهي
- `m-0`, `m-15`, `m-20` - Margin
- `mt-0`, `mb-10`, `mr-15` - Margin اتجاهي

### الألوان
- نصوص: `c-black`, `c-white`, `c-grey`, `c-red`, `c-green`, `c-blue`, `c-orange`
- خلفيات: `bg-red`, `bg-green`, `bg-blue`, `bg-orange`, `bg-white`, `bg-eee`

### الحدود والتقوس
- `rad-6`, `rad-10`, `rad-half` - تدوير الزوايا
- `b-none` - إزالة الحدود
- `border-ccc`, `border-eee` - حدود بلون رمادي

### الخطوط
- `txt-c` - توسيط النص
- `fs-13`, `fs-14`, `fs-15`, `fs-25` - أحجام الخط
- `fw-bold` - خط عريض

### تجاوب الشاشات الصغيرة
- `block-mobile` - عرض كتلة في الموبايل
- `hide-mobile` - إخفاء في الموبايل
- `txt-c-mobile` - توسيط النص في الموبايل

### Overview

**HTML/CSS Template Four** is a complete dashboard template created by **Osama Elzero** as part of his web development tutorial series. Built with pure HTML and CSS (no JavaScript libraries), it features 8 fully designed pages with a custom CSS framework.

### Benefits

- **For Developers:** Learn real project structure, responsive design, Flexbox/Grid, CSS variables, and save time with a ready-made framework
- **For Businesses:** Get a production-ready dashboard UI at zero cost, covering admin panels, settings, user profiles, and reports
- **For Learners:** A complete educational reference with commented code in Arabic, perfect for graduation projects

### Services

| Service | Pages |
|---------|-------|
| **Dashboard Analytics** | `index.html` - KPIs, charts, news, tasks, social media stats |
| **Settings & Security** | `settings.html` - Site control, 2FA, backup, social links |
| **User Profiles** | `profile.html` - User info, stats, activity history |
| **Project Management** | `projects.html` - Project cards, progress, team members |
| **Online Courses** | `courses.html` - Course catalog with prices and enrollment |
| **Friends & Social** | `friends.html` - Friend cards with personal stats |
| **File Manager** | `files.html` - File listing, types, sizes, totals |
| **Subscription Plans** | `plans.html` - Free, Basic, Premium tier comparison |
